# Settings guide

Settings panel opens from the **library** toolbar only. Theme cycles on the library (and site pages) toolbar — not inside the panel, not on the reader.

| Area | Where | Behavior |
|------|-------|----------|
| Language | Settings + library EN/PT chip | UI EN / PT |
| Theme | Library / site toolbar only | light / dark / system |
| Typography | Settings; reader text-size button | font scale, line height |
| Links | Settings + reader **Links** dropdown | density + providers (**wide only**) |
| Library data | Settings | Export / import JSON pack |
| App data | Settings | Export / import settings |
| Notes | — | **None** — Hypothes.is owns notes |
| Updates | Banner + force update (PWA) | Library + reader |
| Maintenance | Settings | Clear site data |

## Export pack (app only)

Includes settings, reading positions, density prefs. **Never** annotation bodies.
