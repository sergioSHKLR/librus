Help

Quick start for the three-pane reader.

Start

  1. Serve the public/ folder (npm start), not the repo root.
  2. Open a book from the library.
  3. Read in the center; annotate with Hypothesis on the left (wide) or standard sidebar (narrow).
  4. On a wide screen, select a term or click a link to consult on the right.
  5. Use Contents to jump sections; search in the main toolbar.
  6. Cycle text size on the reader toolbar.
  7. On wide screens, open Links (link icon) for density and providers (globe / book).
  8. Theme and settings live on the library home. Library button always returns home.

Panes

  • Annotate — Hypothesis notes and highlights.
  • Read — book text, search, contents, typography.
  • Consult — Wikipedia, Wiktionary (wide only).

URL preferences

Query strings write through to localStorage, for example ?lang=en&theme=dark&density=hi&font=1.125&prov=w,d.

Mobile

Narrow screens show the book and notes only — no research links. Prefer landscape desktop for the full experience.

Notes

Log in via Hypothes.is. Export notes from your Hypothesis account. App settings export only preferences (not annotations).
